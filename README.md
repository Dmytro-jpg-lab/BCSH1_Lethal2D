# BCSH1_Lethal2D
BCSH1 - Semestrální práceVarianta: (b) Jednoduchá počítačová hra 
Téma: 2D Top-down Horror Scavenger (inspirováno Lethal Company)

Plánované funkcionality:Hratelnost: 
Průzkum procedurálně generovaných úrovní, sbírání cenných předmětů a jejich transport do bezpečné zóny. 
Nepřátelé (User vs PC): Implementace minimálně 3 typů nepřátel s různým chováním (hlídkování, pronásledování hráče). 
Úrovně: Minimálně 3 různé mapy nebo konfigurace s rostoucí obtížností. 
Plynulá animace: Realizace plynulého pohybu hráče a vizuálních efektů (např. kužel světla baterky). 
Práce se soubory: Ukládání herního progresu (získané finance, odemčené úrovně) do JSON nebo textového souboru. 
Technologie: C#, .NET, Unity/Godot. 

Seznam použitých zdrojů a assetů:(Zde budou doplňovány odkazy na tutoriály, externí assety a AI nástroje podle požadavků zadání)
